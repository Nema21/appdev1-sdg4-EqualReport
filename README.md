# LearnHub — SDG 4: Quality Education

An Angular web application that helps users discover educational books and learning resources using the Open Library API.

---

## SDG Chosen

SDG 4 — Quality Education

This project promotes access to educational materials by allowing users to explore books related to education, learning, and academic growth.

## How to Run Locally

```bash
git clone https://github.com/your-username/appdev1-sdg4-learnhub.git
cd appdev1-sdg4-learnhub
npm install
ng serve
```

Open browser:

```
http://localhost:4200
```
